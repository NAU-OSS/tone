# *tone*
*tone* is a lightweight, command-line tool for hobby audio production and experimentation. This software is designed to be modular and encourages private forks and modifications for personal features and preferences.

## Requirements
- Unix-like system
- Bash
- Audio input/output devices

## Getting Started
- Clone the repository to your local device
- Make modifications and push to remote
- Create a pull request
- Expect feedback! Modify the same PR and tag the reviewer in the comments
